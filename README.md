Breast Cancer Detection Using Random Forest
Project Overview

This project presents a Machine Learning based Breast Cancer Detection System using the Random Forest Classifier algorithm. The system predicts whether a tumor is malignant or benign using medical diagnostic features from the Wisconsin Breast Cancer Dataset.
Dataset Information

Dataset Used:

Wisconsin Breast Cancer Dataset

The dataset contains:

30 numerical input features
1 target variable (Diagnosis)

Target Labels:

M = Malignant
B = Benign

Machine Learning Algorithm
Random Forest

Random Forest is an ensemble learning algorithm that combines multiple decision trees to improve prediction accuracy and reduce overfitting.
